Montreal Crime Data Analysis (2015–2021)

This repository contains an exploratory data analysis (EDA) of crime reports in Montreal. The goal of this project is to identify geographical hotspots, categorise prevalent crimes, and track year-over-year trends to see if crime rates are increasing or decreasing.
Workflow & Features:

• Data Cleaning: Handled unwanted columns, managed date-time formatting, and verified data integrity (0 missing values found).

• Temporal Analysis: Visualised crime trends from 2015 to 2021, revealing that 2015 had the highest volume of cases while 2021 saw the fewest.

• Geospatial Insights: Identified the top 10 cities where crimes are most frequently reported.

• Visualisations: Utilised Matplotlib, Plotly, and Seaborn for interactive and static charts, including "Top Words" in crime categories.

Tech Stack:

• Language: Python
• Libraries: pandas, NumPy, Matplotlib, Seaborn, Plotly
